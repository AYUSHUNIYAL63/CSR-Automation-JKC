# CSR-Automation-JKC

AI-powered CSR compliance automation for Jan Chetna Kendra, Dehradun.

## Problem
NGOs waste 5+ hours manually checking CSR donations against Schedule VII.

## Solution  
Google Sheets + Gemini AI system:
- OCR from cheque images & PDFs
- Flags high-risk funds: Political, Personal accounts, Non-Schedule VII
- 40 seconds vs 5 hours manual review
- Saves 95% effort

## Tech
Google Apps Script, Gemini 1.5 Pro, Drive API

## Impact
₹3.5L+ high-risk funds flagged in pilot
