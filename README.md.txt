# CSR-Automation-JKC

AI-powered CSR compliance automation tool for Jan Chetna Kendra, Dehradun.

## Problem
Indian NGOs waste 5+ hours manually checking if CSR donations violate Schedule VII of Companies Act, 2013.

## Solution
Automated Google Sheets + Gemini AI system that:
- Extracts data from cheque images & PDFs using OCR
- Analyzes transactions against CSR law via Gemini 1.5 Pro
- Flags high-risk funds in 40 seconds vs 5 hours manual
- Saves 95% effort for compliance teams

## Tech Stack
Google Apps Script, Gemini 1.5 Pro, Google Sheets, Drive API

## Impact
₹3.5L+ high-risk CSR funds flagged in pilot run
